A Better Setlist

A Better Setlist solves a real-world problem my band faces: having to make and re-make setlists for every show and manually check that our chosen songs won't go beyond our allotted time.

Intended for mobile use, it stores your songs persistently to your device to allow setlists to be made quickly and easily. It also has a PDF export feature, to facilitate printing copies to take on stage.

I also wanted to practice data visualization, so I added a feature using Recharts (based on D3) to view graphs of the setlist to see how the song intensity and crowd response changes as the show progresses.

ScoreHoard

ScoreHoard is an API that keeps track of stats and scores for video game makers. Version 1.0.0 allows Add Requests (to add usernames and scores into our database) and Leaderboard Requests (to retrieve stored data for your game).

In this four-person project, I worked primarily on the back-end (app.js) in a pair-programming setup, and I also developed the front-end interface for the API in the game demo featured on the website.
